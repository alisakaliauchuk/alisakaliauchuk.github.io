Mes compétences

J'ai appris de nombreux langages de programmation et les outils qui m'ont permis d'avancer davantage et qui m'ont beaucoup aidé dans les projets pendant mes études.

Développement

HTML

HTML - est un langage utilisé afin de créer et de représenter le contenue d'une page web et sa structure.

CSS

CSS - est un langage utilisé dans HTML ou XML pour mettre en forme les pages web.

XML

XML - est un langage de balisage "autodescriptif" et il est similaire au HTML.

RSS

Flux RSS - est un fichier texte XML qui contient des détails sur les dernières mises à jour d'un site web (titre, description et lien vers le contenu).

PHP

PHP - est un langage de programmation libre, largement utilisé dans le développement de sites web.

PostgreSQL

PostgreSQL - est un système de gestion de base de données.

JavaScript

JavaScript - est un langage de programmation de script, principalement utilisé pour produire des pages web dynamiques.

JSON

JSON - est un format texte permettant de stocker et d'échanger des données de manière lisible par l'utilisateur et analysable par la machine.

Java et JavaFX

Java (et JavaFX) - est un langage de programmation des applications, et JavaFX est un framework complémentaire à Java.

Des outils / logiciels

Apache

Apache - est un logiciel de serveur web Open Source le plus populaire.

VirtualBox

VirtualBox - est un logiciel de virtualisation multiplateforme open source, qui permet de créer et d'exécuter des machines virtuelles.

Looping

Looping - est un logiciel de modélisation conceptuelle de données gratuit.

Visual Studio Code

Visual Studio Code - est un logiciel pour éditer le code source léger, mais puissant, lequel on peut exécuter sur bureau.

WordPress

WordPress - est un système de gestion de contenu gratuit pour création des sites web.

GitHub

GitHub - est une plateforme basée sur le cloud où vous pouvez stocker, partager et travailler avec d'autres pour écrire du code.

WireShark

WireShark - est un analyseur de réseau open source, capable de capturer et d'afficher les détails du trafic réseau en temps réel.
